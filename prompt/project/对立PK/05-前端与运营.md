# 对立PK：前端与运营

## 用户侧页面

### PK 列表页

展示所有永恒 PK 话题。

建议字段：

- 话题标题。
- A/B 阵营名称。
- 当前阶段。
- 当前热度对比。
- 下注人数。
- 倒计时。
- 上局胜方。
- 守擂/翻盘状态。

### PK 详情页

详情页是核心玩法页。

建议区域：

- 顶部：话题标题、A/B 阵营、当前阶段、倒计时。
- 中部：对抗条、热度数字、赔率、下注按钮。
- 下部：评论战场，A/B 双列。
- 侧栏或折叠区：历史战绩、赛季战绩、最近对局。

### 我的参与状态

用户进入详情页后需要明确展示：

- 我本局是否已下注。
- 我选择的阵营。
- 当前预计收益或赔率。
- 本局是否可继续互动。
- 结算后我的输赢与 payout。

## 互动战场

### 评论

评论发布时必须选择或继承阵营。

建议规则：

- 已下注用户默认使用下注阵营。
- 未下注用户在锁局期仍可评论，但必须选择阵营。
- 冷却期禁止评论。

### 双列展示

A/B 评论双列展示：

- A方评论只显示 A 阵营评论。
- B方评论只显示 B 阵营评论。
- 回复跟随父评论所在阵营。
- 每条评论展示热度贡献、点赞数、回复数、拉踩数。

### MVP 评论者

P1 实现。

口径：

- 当前局单条评论热度最高者。
- 或当前局累计评论热度最高用户。

## 运营侧能力

### 话题维护

运营需要维护：

- 标题。
- A/B 阵营名称。
- 封面或阵营图。
- 排序。
- 状态：启用/停用。

默认种子话题来自规则文档：

- 足球GOAT之争：梅西 vs C罗。
- 女团门面之争：张元英 vs 柳智敏。
- 未来超级大国：中国 vs 美国。
- 手机之王：iPhone vs 安卓。
- 超级英雄宇宙：漫威 vs DC。
- 电竞GOAT：Faker vs Uzi。
- 人类最好的伙伴：猫 vs 狗。
- 豆腐脑之争：甜 vs 咸。
- 学历vs能力：学历 vs 能力。
- 华语乐坛之王：周杰伦 vs 林俊杰。
- NBA历史地位：詹姆斯 vs 库里。
- 世界之都：纽约 vs 伦敦。
- 新能源汽车之王：特斯拉 vs 小米。
- 手机操作系统：安卓 vs iOS。
- 外卖平台大战：美团 vs 饿了么。
- 短视频霸主：抖音 vs 快手。

### 规则配置

P0 可硬编码，P1 建议运营可配置：

- 单票价格，默认 100。
- 下注期小时数，默认 48。
- 锁局期小时数，默认 24。
- 冷却分钟数，默认 10。
- 赛季天数，默认 30。
- 评论单条热度上限，默认 20。
- 下注热度系数，默认 0.5。
- 虚拟底池。

### 风控运营

P1 建议提供：

- 用户风险系数调整。
- 回合热度重算。
- 评论质量分修正。
- 异常动作列表。

## 视觉效果分级

### P0

- 清晰的 A/B 对抗条。
- 阶段状态和倒计时。
- 已下注状态。
- 双列评论战场。
- 历史战绩卡片。

### P1

- 龟势PK火焰文字。
- VS 球与弹簧动画。
- 评论发送后对抗条震动。
- MVP 头像展示。

### P2

- 粒子背景。
- 阵营特殊皮肤。
- 连胜与翻盘特殊动效。

视觉效果不应阻塞 P0，因为核心价值是回合、下注、热度、结算闭环。
